落氧节拍器

💡 核心机制

  • 绝对精准：采用 Web Audio 硬件时钟调度，在高密集状态下不卡顿不减速。
  • 自动切白：导入音频时，系统会自动扫描并删除开头振幅低于阈值的静音部分，确保音乐与第一拍完美咬合。

🎵 编排语法说明 (每行一个指令)

  1. 标准指令：BPM 拍号。例如 140 5/4，表示以 140 速度播放一小节 5/4 拍。
  2. 省写BPM：拍号。例如 6/4，将继承上一行的速度。
  3. 省写拍号：BPM。例如 130，将继承上一行的拍号。
  4. 重复小节：指令×次数。例如 130 4/4×4 或 7/4×2，连续重复该小节 N 次。
  5. 复合多拍号 (高级)：用 + 连接。例如 140 3/4+4/4×2，表示在 140 速度下，先走一小节 3/4，紧接着走两小节 4/4。

🚀 快速上手步骤

  1. 在“当前状态”中设置基本参数，点击“单节拍播放”即可作为普通节拍器使用。
  2. 若要使用曲目编排，在文本框编辑好指令后，必须先点击“解析编排”。
  3. 若要带伴奏练习，先导入本地音频文件，待提示“处理完成”后，点击“音乐+节拍同步播放”。
