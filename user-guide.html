📚 准备曲库

网站启动后默认进入“曲库”选项卡。你可以选择导入内置的在线曲库，或者按照指定格式导入自己的本地曲库。

1.1 导入在线曲库

点击橙色的 导入在线曲库 按钮，系统会自动从 GitHub 拉取最新的内置曲库文件。导入成功后，下方会显示曲库包含的游戏范围数量和总曲目数。

网络要求
在线导入需要科学上网环境，若导入失败请检查网络连接，或改用本地导入。

1.2 导入本地曲库

点击 导入本地 按钮，在弹窗中输入曲库名称（可选）、允许的范围标签（必须，一行一个），并选择本地的 .txt 曲库文件。

TXT 文件格式要求：

  • 一行 [游戏名] 作为范围声明。
  • 下面逐行写曲名，可以不带序号，系统会自动去除前缀序号。
[Phigros]
1.Distorted Fate
2.DESTRUCTION 3,2,1
3.Glaciaxion
[Arcaea]
1.Sheriruth
2.Arcahv
范围标签冲突
如果你自定义的范围标签与内置标签同名（如也叫 Phigros），会导致数据冲突。导入本地时请务必声明该曲库包含哪些标签，系统只会识别声明的标签。

1.3 曲库管理与编辑

在曲库下拉框旁，有 重命名 和 删除 按钮（内置在线曲库不可重命名/删除，只能清空内容）。

直接编辑曲库：在游戏范围列表中，每个范围右侧有一个 ✎ 按钮。点击后会弹出编辑窗口，你可以：

  • 搜索：在顶部搜索框过滤曲目。
  • 删除：点击曲目右侧的 X 删除。
  • 添加：在底部输入新曲名并点击“添加”。
  • 保存/重置：点击“保存修改”会将变更存入浏览器本地；点击“重置回初始状态”会撤销未保存的修改。

⚙️ 配置出题序列

2.1 选择范围与过滤

导入曲库后，在“选择游戏范围”卡片中勾选你要参与抽取的游戏（如 Phigros, Arcaea）。

曲目过滤功能：你可以对选中的范围进行进一步筛选：

  • 字符类型：限制曲名必须包含（或不包含）字母、日文、汉字，或允许其他字符。
  • 最少字符长度：过滤掉名字太短的歌（包含空格）。

2.2 抽取设置

系统提供两种抽取模式：

  • 总数量模式：输入一个总数，系统会从你选中的所有范围里均匀、随机地抽取指定数量的曲目。
  • 定制数量模式：为每个选中的游戏范围分别指定要抽取的数量。

配置好后点击 生成曲目列表，系统会自动跳转到“手动输入”选项卡，并填入生成的曲目。

2.3 手动输入与正则选曲

你也可以直接在“手动输入”选项卡的文本框中粘贴曲名（每行一首，无需序号）。

从曲库手动选曲：在“曲库”选项卡中点击 从曲库手动选曲...。

  • 支持正则表达式搜索，例如输入 ^A.* 可以搜索所有以 A 开头的歌。
  • 点击曲目行即可选中，选完后点击“导入到出题序列”。

排列选项：你可以将当前文本框中的曲目进行“随机排列”或“按长度排列”。点击“应用排列”生效。

最后，可以在“备注信息”中填写如“范围：Phigros, Arcaea”等提示信息，点击 解析并开始游戏 正式进入游戏界面。

🎮 游戏进行中

3.1 计时器与基础控制

页面顶部显示巨大的计时器。你可以点击 暂停 暂停计时，点击 结束游戏 进入结算页面。

3.2 开启字符 (核心功能)

在“开启字符”输入框中，输入你想要公开的字母（如 aeiou）。输入后，下方所有曲目中对应的字母都会高亮显示，未开出的字符会变成遮罩符号（默认 ?）。

系统会自动处理全角/半角字符等效（如开 ( 会同时开出 （），并且忽略大小写。

下方有三个统计数据卡片：曲名数（剩余可见）、已开字符（你输入了几个字符）、揭示率（所有字符中被开出的比例）。

3.3 曲目交互操作

在“解析结果”列表中，每首歌右侧有两个小图标按钮：

  • 查询来源 (放大镜图标)：如果开启了“曲目来源查询”设置，点击此按钮可查看该曲目属于哪个游戏范围。
  • 单独提示 (太阳图标)：点击会弹出一个模态框，展示该曲目的所有字符。你可以点击任意字符将其强制单独公开（显示为黄色），这不会影响全局的已开字符。适合作为特殊提示。

显示全名与隐藏：直接点击曲目行，会显示完整的曲名（绿色高亮）。当玩家猜对后，你可以点击 隐藏已猜出 按钮，将所有处于“显示全名”状态的曲目从列表中移除（移入隐藏状态）。隐藏的数量会显示在按钮左侧。

3.4 撤销与重做

在“开启字符”输入框右侧，有撤销和重做按钮。每当你修改了开字符、隐藏了曲目或单独提示了字符，系统都会存入历史记录。你可以随时撤销误操作。

点击 重置 会清空所有开启的字符和单独提示。

🏁 结算与导出

点击“结束游戏”后，会进入结算页面。页面包含三个主要部分：

  • 结算信息：包含用时、曲目总数、猜出数量、开启字符数、揭开率等详细数据。
  • 未猜出的曲目：列出玩家没能猜出来的歌曲全名。
  • 完整题面：本局游戏的所有曲目全名及备注，用于复盘。

每个部分都有 复制 和 导出图片 按钮。导出图片功能会将文本生成为一张图片，在手机端会弹出图片供长按保存，在电脑端会直接下载 PNG 文件。

结算完毕后，可以选择“开始新游戏”或“返回编辑”。

🛠️ 全局设置详解

点击右上角的齿轮图标打开设置面板。所有设置会自动保存在浏览器本地。

遮罩符号

默认是 ?。你可以改成任意单个字符，如 * 或 _。

开字母限制

  • 最大开字符数：限制全局最多能开几个字符，防止直接把歌名开完。设为 0 为不限制。当玩家尝试输入超出限制的字符时，会被拦截并提示。
  • 最大揭开率限制：限制当前所有曲目被揭开的百分比上限。达到上限后无法继续开新字符。

功能开关

  • 开启计时功能：关闭后游戏界面将不显示计时器。
  • 开启结算功能：关闭后点击“结束游戏”会直接返回首页，不生成结算报告。
  • 空格也隐藏为遮罩符号：默认情况下空格正常显示，开启后空格也会变成 ?，增加难度。
  • 自动根据游戏范围填充备注：生成曲目或开始游戏时，自动在备注栏写入“范围：XXX, YYY”。
  • 开启字符排序：将你输入的乱序字符按“数字-大写字母-小写字母-日文-汉字”自动排序显示在顶部。
  • 开启曲目来源查询：在游戏界面每首歌旁显示放大镜按钮，用于查询来源。
  • 固定导出图片 9:16 比例：导出图片时强制裁剪/填充为适合手机壁纸的比例。

导出图片颜色设置

你可以自定义导出图片的背景色和文字颜色，以适应不同的分享平台背景。
